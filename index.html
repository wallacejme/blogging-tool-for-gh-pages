Kinds of content styling

Typography, section dividers

Text emphasis: bold, italic, bold and italic and strikethrough. We can also have inline quotations, which might be achieved by using a <q> element or explicit quotations marks, like "double quotes" and 'single quotes'.

Explicit quotation marks might be converted to curling quotes if some preprocessing is applied to the post (not happening yet). <q> elements generate curling quotes by default through CSS rules.
